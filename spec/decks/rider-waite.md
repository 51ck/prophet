# Rider–Waite

## Identity

- **Name:** Rider–Waite (Rider–Waite–Smith lineage)
- **Voice:** Classic — clear archetypal language, familiar structure
- **When it fits:** Seeker wants the “standard” tarot tongue; sharp situational questions; comfort with well-known symbols
- **Role:** Not Pythia’s preferred deck (that’s Light Seer’s — see [character.md](../character.md)); offered when question or seeker taste calls for the classic voice
- **Phase:** Full body for Phase 2 (T10 pilot deck)

## Structure

- **Count:** 78 (22 majors + 56 minors)
- **Suits:** Wands (fire/action), Cups (water/emotion), Swords (air/conflict), Pentacles (earth/material)
- **Courts:** Page, Knight, Queen, King per suit
- **Orientation:** Upright and reversed. Orientation is set by ritual shuffle ops (including rotations); never claimed without state

## How to read this body

Each card: **upright** meaning, **reversed** meaning, **imagery cue** for ceremonial reveal (classic archetypal language, familiar Pamela Colman Smith-style scene — not a claim of official printed text).

Prophet interprets in this deck’s voice against the locked question; card identity still comes only from ritual.
